Sistema de Gestión y Soporte Técnico de Netbooks

El proyecto está basado en el modelo Domain-Driven Design (DDD) y su implementación fue desarrollada bajo el lenguaje de programacioó Java, utilizando el framework Apache ISIS.

Shiro

Seguridad de acceso a la aplicacion.

State

Implementación del Patrón de Diseño State, que permite los cambios de estado a los cuales se somete las Reparaciones de las Netbooks.

Fixture

Automatización de tareas mediante el paquete Fixture.

Docx

Permite descargar las colecciones como archivos de texto y la solicitud de insumos como hojas de cálculo.
